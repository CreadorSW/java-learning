# Contributing to this project

¡Gracias por interesarte en colaborar! Para mantener la consistencia en los documentos y facilitar el flujo de trabajo, por favor sigue estas instrucciones.

## Configuración del Entorno

Para trabajar en este repositorio, necesitás instalar Typst en tu sistema:

+ Visitá la sección de instalación en [github.com/typst/typst](https://github.com/typst/typst) para ver las instrucciones específicas de tu sistema operativo.
+ Es importante que consideres la siguiente recomendación oficial:

> If you prefer an integrated IDE-like experience with autocompletion and instant preview, you can also check out our free web app. Alternatively, there is a community-created language server called Tinymist which is integrated into various editor extensions.

## Flujo de Trabajo

Para hacer pull requests:

+ Hacer un **fork** del repositorio.
+ Crear una rama nueva para tu contribución: `git checkout -b mejora-o-correccion`.
+ Realizar los cambios en el archivo `.typ`.
+ Asegurarse de que el documento compile correctamente localmente antes de hacer el commit.
+ Enviar un **Pull Request** detallando los cambios introducidos y por qué son necesarios.
